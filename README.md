# Installation
 
Naniwa can be built with the following command:
```bash
git clone https://github.com/q-group-work/naniwa.git
cd qulacs
pip install .
```

# Test
 
 Tests can be run using pytest under the 'naniwa' directory.
```bash
pytest
```
